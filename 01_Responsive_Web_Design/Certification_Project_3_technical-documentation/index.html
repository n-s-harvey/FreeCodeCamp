Getting Started with C++

The first thing to understand about C++ is that it is a high-level language with low-level features.

This means that you can expect to encounter many issues, such as SEGFAULT and off-by-one errors.

Finding your Favorite C++ Error

One of the best ways to encounter errors in C++ is by using pointers.

Pointers are an enigma.

C++ Data Types

There are different kinds of objects in C++, some of which are in the standard library. All of them are easy to work with and easier to break your code.

Here are some different ones:

  • int
  • long int
  • long long int
  • unsigned int
  • unsigned long int
  • unsigned long long int

Troubleshooting your C++ Code

A serious piece of advice: If your C++ code does not work properly, Google the first part of the error. Don't try to search the whole thing.

Building your Code

If you are trying to program with C++ on Windows, you are better off not doing that.

I recommend installing Ubuntu or your favorite Linux distribution. It is much easier to install the compiler packages there than it is on Windows.

About challengerCodeCamp

Thank you for your interest in the challengerCodeCamp!

We offer different camps to empower you in your programming journey.

We have three in-person code camps available: C++, HTML and CSS, and Java.

C++ Code Camp

The C++ Code Camp will teach you how to create your own fully-functional segfaults.

HTML & CSS Code Camp

The HTML & CSS Code Camp will teach you the basics of markup and prepare you to enjoy SASS.

Java

The Java Code Camp will teach you just enough to be dangerous with the world's most-used (and disliked) language.
